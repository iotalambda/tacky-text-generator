# Example GIF Summaries

These summaries describe the tacky WordArt-style 3D text animations for reference when building similar effects.

## 1. welcome_to_my_mediocre_blog.gif

**Text:** "Welcome to My" (line 1), "Mediocre Blog" (line 2)

**Visual Style:**
- **Font:** Bold italic serif font (similar to Times New Roman Bold Italic)
- **Face Color:** Purple/violet gradient
- **Extrusion/Side Color:** Deep maroon/burgundy red
- **Outline:** Dark purple/black thin outline around letters
- **Z-Depth:** Heavy extrusion (~15-20% of letter height), giving strong 3D depth

**3D Perspective:**
- Slight isometric angle, text tilted back
- Extrusion extends down-left, creating shadow-like depth effect
- Text appears to float in space with perspective distortion

**Animation:**
- Gentle rocking/swaying motion (left-to-right oscillation)
- Text rotates slightly around Y-axis creating a swinging pendulum effect
- Smooth loop with ~2-3 second cycle
- Multi-line text moves together as a unit

**Material/Lighting:**
- Matte finish on face
- Darker shading on extruded sides
- Light source from upper-right

## Common Patterns for WordArt-Style 3D Text

### Color Combinations (Face / Extrusion):
1. Purple / Maroon
2. Lime Green / Dark Green
3. Yellow / Blue (high contrast)
4. Light Blue / Dark Blue (monochromatic)
5. Blue / Gold (royal)
6. Red / Black
7. Orange / Purple
8. Pink / Hot Pink
9. Silver / Black

### Font Styles:
- Bold italic serif (Times New Roman Bold Italic)
- Heavy sans-serif (Impact, Arial Black)
- Decorative/display fonts
- Always bold/heavy weight

### Animation Types:
1. **Y-axis rotation** - Swinging/pendulum motion
2. **Wave/undulate** - Letters move up/down in wave pattern
3. **Wobble/shake** - Quick oscillating vibration
4. **Individual letter spin** - Each letter rotates independently
5. **Zoom pulse** - Text scales in/out slightly
6. **Bounce** - Springy up/down motion

### 3D Effects:
- Extrusion depth: 10-25% of letter height
- Perspective distortion (vanishing points)
- Wave/arc text paths
- Isometric or slight tilt angles

### Material Characteristics:
- Matte or glossy faces
- Contrasting side colors
- Outline/stroke around letters
- Gradient fills on faces
